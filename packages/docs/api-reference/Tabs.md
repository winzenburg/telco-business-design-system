# Tabs

A reusable UI component from the Comcast Business Design System.

## Import

```tsx
import { Tabs } from '@comcast/design-system';
```

## Usage

```tsx
<Tabs />
```

- Keyboard navigation support
- Screen reader compatibility
- Focus management
